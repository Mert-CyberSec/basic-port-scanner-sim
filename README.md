# Basic Network Scanner Simulation

A Python-based interactive script that simulates a network port scanning process with authentication. Built as a foundational exercise in cybersecurity tool development.

## Features
- User authentication and access control simulation (`if/else` logic)
- Interactive targeting (IP and Port inputs)
- Automated loop-based scanning output (`for` loops and `time` module)

## Usage
Run the script using Python 3:
```bash
python kamp1.py

(Enter "root" when prompted for the authorization password).
